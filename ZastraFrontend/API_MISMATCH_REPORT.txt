API Endpoint Alignment Analysis (Frontend vs Backend)
===================================================

Good news! Nearly 100% of the frontend services map exactly to the backend controllers. 
The routing, paths, and HTTP methods match correctly across Projects, Profile, Portfolio, Integrations, Gamification, Directory, and Auth.

However, there is ONE minor change required in the frontend to achieve complete alignment.

1. Missing Endpoint in `activityService.js`

   Frontend Service: `src/services/activityService.js` currently defines:
     - `getGlobalStats`
     - `getGithubStats`
     - `getContestStats`
   
   ACTION REQUIRED:
   You need to add the heatmap API call to `src/services/activityService.js` so the frontend can retrieve the activity heatmap data.

   Add this line to `activityService.js`:
     getHeatmap: () => api.get('/api/v1/activity/heatmap'),
